Map 01
Hispanic Population Distribution in the United States

To solve this problem we get data from the U.S Census Bureau, use the project to convert the file to the NAD 1983 Contiguous USA Albers projected coordinate system. Then we select and export the 48 Contiguous United States to create a new shapefile. Now we touch on the symbology, for a dot density map we change the primary symbology to dot density and for a choropleth map we change it to graduated colors. Afterwards we change the field to whatever data is needed then create a layout where we add the legend, scale and north arrow to make it look professional. With all of that we can answer the question in Part II.

Map 02
Spatial Distribution of the Density of Airbnb Reviews (per square mile) in Chicago by Police District

To solve this problem we used data on crimes in Chicago and airbnb data. First we changed the map to the appropriate UTM coordinate system. Then we specifically select data that mentioned homicide. Then by using the summarize we calculated the number of homicides and Airbnb reviews, and average price, for each police district. After that was done We used calculate field to calculate the density of homicides and density of Airbnb reviews for each police district. Lastly we created choropleth maps.

Fig. 3
Map 03
Best Area for a New Store Selling Healthy Foods

To solve this problem first I had to project all the data to NAD 1983 StatePlane Pennsylvania South FIPS 3702 (US Feet). Then I buffered and unioned the regional rail stations and the subway stations then we intersected it with the empowerment zones naming it inclusive. Then I buffered and unioned the farmers markets and the corner stores and named it exclusive. Next I erased the exclusive layer from the inclusive layer and named it candidates. Afterwards I cleaned up the candidates layer using the multipart to single part tool. Lastly I calculated the area of each candidate. With all of that we can answer the question in part II.
